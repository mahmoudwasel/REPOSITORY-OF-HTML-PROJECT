Definition of database

database or electronic database is defined as, any collection of data, or information, that is specially organized for rapid search and retrieval by a computer ,databased is used to facilitate the storage, retrieval, modification, and deletion of data in conjunction with various data-processing operations. database is stored as a file or a set of files. The information in these files may be broken down into records, each of which consists of one or more fields. this file is defined as basic units of data storage.to reach information database records and files must be organized .query are the main way which users can use to reach information, query is defined as a computer programming language used to retrieve information from a database.database is used in many applications as airline reservations, production management functions, medical records in hospitals, and legal records of insurance companies. The largest databases are used in many applications as governmental agencies, business organizations, and universities, large database contains texts of such materials as abstracts, reports, legal statutes, wire services, newspapers and journals, encyclopaedias, and catalogs of various kinds.(2)
